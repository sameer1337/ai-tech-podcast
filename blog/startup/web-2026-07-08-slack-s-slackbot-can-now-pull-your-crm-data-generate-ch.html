According to VentureBeat, Slack's Slackbot can now pull CRM data, generate charts, and send DocuSigns, all from a chat message. This integration is a significant step forward for the two companies, which have been working together since Salesforce acquired Slack for $27.7 billion. As reported by VentureBeat, the mechanism behind this expansion is a set of dedicated Model Context Protocol (MCP) integrations.

This development is crucial for businesses that rely on both Slack and Salesforce, as it streamlines their workflow and reduces the need for manual data entry. With this integration, users can access a wide range of Salesforce features, including CRM data, Tableau analytics, and Data 360 customer profiles, all through a single conversational prompt. As noted by VentureBeat, this is a significant improvement over traditional methods, which often required manual updates and migrations.
